# Análise de Dados de Marketing - Power BI

Este projeto consiste em uma análise exploratória de dados de marketing para identificar padrões de comportamento de consumo, eficácia de campanhas e perfil demográfico dos clientes.

## 2. Objetivos
* Analisar o perfil de renda e escolaridade dos clientes.
* Identificar quais países possuem maior volume de compras.
* Avaliar o desempenho das campanhas de marketing (Campanhas 1 a 5).
* Verificar a correlação entre presença de filhos em casa e gastos por categoria.

## 3. Tecnologias Utilizadas
* **Power BI**: Para ETL, modelagem de dados e visualização.
* **DAX**: Para criação de métricas personalizadas.
* **CSV**: Fonte de dados original.

## 4. O que você encontrará no Dashboard
1. **Visão Geral**: KPIs de vendas, total de clientes e ticket médio.
2. **Comportamento de Compra**: Análise de compras via Web, Catálogo e Loja Física.
3. **Geolocalização**: Distribuição de vendas por país.

## 5. Como visualizar
Para abrir o arquivo `.pbix`, você precisará do [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
Alternativamente, veja os prints na pasta `/screenshots`.
